## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn
```

## Env file example

```
# app config
PORT=
DEBUG=

# app secrets
JWT_TOKEN=testejwt

# database
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_PORT=
DB_DATABASE=
```

## Running the app

```bash
# init docker container
$ yarn docker:up

# watch mode
$ yarn dev

```
